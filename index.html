Luas Segitiga

Rumus Luas Segitiga yaitu :

L = 1/2 x a x t

Dimana:

L = Luas

a = Panjang Alas

t = Tinggi

Keliling Segitiga

Rumus Keliling Segitiga yaitu :

K = S1 + S2 + S3

Dimana:

K = Keliling

S1 = Sisi A

S2 = Sisi B

S3 = Sisi C

Luas Jajar Genjang

Rumus Luas Jajar Genjang yaitu :

L = a x t

Dimana:

L = Luas

a = Alas

t = Tinggi



Hitung Luas Jajar Genjang


Keliling Jajar Genjang

Rumus Keliling Jajar Genjang yaitu :

K = 2a + 2b atau K = a + b + c + d

Dimana:

K = Keliling

a,b,c,d = Panjang masing-masing sisi

a = c dan b = d
